Internet Connectivity & WWW

An overview of different types of internet connectivity and the basics of the World Wide Web.

Types of Internet Connectivity

Dial-up Connections

Allows connection via a local server using a standard 56k modem. The PC dials a phone number provided by the ISP to connect to the internet.

Leased Line Connection

A secure, dedicated, and expensive level of Internet connection that keeps the computer online 24/7.

ISDN

Stands for Integrated Services Digital Network, utilizing phone lines to carry digital instead of analog signals.

  • Basic Rate Interface (BRI)
  • Primary Rate Interface (PRI)

DSL (Digital Subscriber Line)

A broadband connection over ordinary telephone lines with various types available, such as ADSL, SDSL, HDSL, RDSL, VDSL, IDSL.

Cable connections

Uses cable TV lines for broadband Internet connectivity, utilizing different channels for upstream and downstream transmission.

VSAT (Very Small Aperture Terminal)

An earthbound station in satellite communications for data, voice, and video signals, but excluding broadcast television.

Wireless Internet Connection

Uses radio frequency bands to connect to the internet, offering high speed via WiFi or Bluetooth.

World Wide Web (WWW)

WWW, developed initially by Tim Berners-Lee at CERN, is a globally distributed internet service that provides seamless access to a wide range of information through a standard set of naming and access conventions.

Architecture

The WWW operates on a distributed client/server model with sites hosting documents accessible through browsers.

Principles of WWW

  • No central control
  • Uniform protocols
  • HTTP as the backbone protocol

Application Layer Protocols

  • HTTP: Defines client-server communications
  • Domain Name System (DNS)
  • File Transfer Protocol (FTP)
  • Simple Mail Transfer Protocol (SMTP)
  • TELNET

Web Browser Features

Includes bookmarking, private browsing, extensions, and more, with support for multiple tabs, back/forward navigation, refresh/reload, stop, home button, and integrated search.
