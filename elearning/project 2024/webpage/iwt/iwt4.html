ELECTRONIC MAIL (E-Mail)

Internet provides main communication services that are E-Mail, Newsgroup (Usenet).

E-mail is a private communication between two parties who have accounts on the internet.

Advantages:
  • It is faster, easier.
  • E-mail is more convenient than traditional mail.
  • E-mail is cheap.
  • It is flexible.
E-mail Networks:

Data and messages can be transmitted from one computer to another using telephone lines, microwave links, communication satellites, or other telecommunication equipments.

E-Mail clients:

If you are accessing the internet at home through an ISP, you will need a program which can send and receive E-mail. This is known as E-mail client.

Protocols of E-mail
1- POP3 (Post Office Protocol 3)

Defines how e-mail messages are to be retrieved from a mail server. POP3 is one of the two protocols that a PC mail should be able to use if it is to send and receive e-mail over the internet.

2- SMTP (Simple Mail Transfer Protocol)

This is the method protocol computers use to send messages by message transfer agents (MTA) on the internet.

3- MIME (Multipurpose Internet Mail Extension)

It represents a standard for describing data types. It was designed as an extension to the simple mail transfer protocol allowing people to send binary data.

4- IMAP (Internet Message Access Protocol)

It is a method of accessing e-mail or bulletin board messages that have been kept on a mail server. IMAP4 supports additional features like searching through e-mail messages for keywords while the messages are still on the mail server.
